# 📚 AI Study Helper

An AI-powered web app that summarizes PDF documents using Google Gemini AI.

---

## 🚀 Features
- Upload PDF
- Extract text automatically
- AI-powered summarization
- Clean Streamlit UI

---

## 🛠️ Tech Stack
- Python
- Streamlit
- PyPDF2
- Google Generative AI (Gemini)

---

## 📸 How it works
1. Upload a PDF
2. Click "Generate Summary"
3. Get AI-generated summary instantly

---

## ⚙️ Setup Instructions

```bash
pip install -r requirements.txt
streamlit run AI_Study_Helper.py
